


# KLCP Practice Exam — 200 MCQs

A collection of 200 multiple choice questions to help you prepare for the **Kali Linux Certified Professional (KLCP)** exam.

---

## Topics Covered

- Kali Linux history (WHoppiX → WHAX → BackTrack → Kali)
- Linux fundamentals (kernel, user space, device files)
- Kali installation and partitioning
- Encrypted installations (LUKS, LVM)
- APT and dpkg package management
- Upgrading and purging packages
- Kernel recompilation and custom builds
- Live-build and custom ISO creation
- Metapackages and tool categories
- USB persistence
- Networking, iptables, and NAT
- Kali on ARM / Raspberry Pi (hostapd, dnsmasq)
- Forensics mode
- Security assessment concepts (CIA triad, vulnerability types, risk)
- Documentation and help resources (man pages, apropos)
- Custom repository creation (reprepro, GPG signing)

## How to Use

1. Open `KLCP_200_MCQ.md`
2. Work through the questions without looking at the answer key
3. Check your answers in the **Answer Key** table at the bottom of the file
4. Review any topics where you scored low
